hive安装

发表于 2020-09-25 更新于 2021-06-15 分类于 大数据

Hive

上一篇安装了hadoop，这次我们来使用hive操作haddop。

环境

  • macOS Catalina 10.15.6
  • JDK 1.8.0_261
  • homebrew

安装

  • brew install hive

配置

  • 环境变量

  • hive-site.xml

    cd /usr/local/Cellar/hive/3.1.2_1/libexec/conf，没有就：mv hive-default.xml.template hive-site.xml，然后添加如下configuration

  • 配置元数据库存

    • 装好mysql，创建数据库metastore。
    • 下载mysql驱动放到hive的lib目录下：https://dev.mysql.com/downloads/file/?id=496588
    • 到bin目录下，schematool -initSchema -dbType mysql

运行

  • 启动metastore：hive –service metastore &
  • nohup hiveserver2 & (若需要jdbc连接)
  • 运行：hive
